      vue3 + electron+vite 写桌面应用
      现在需要搞一个config.ini 配置文件
      内容如下：
      [ % General]
      config = 1 是否启动配置功能
      logLevel = 1 LOG打印级别
      maxLine = 8 日志显示最大行数

      [serialPort]
      com = COM10 串口号
      baud = 115200 波特率
      dataBit = 8 数据位
      parity = None 效验位
      stopBit = 1 停止位
      使得打开桌面应用，就能读取文件，然后将相关内容，展示在页面上
